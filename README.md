# AuthShield 🔐

AuthShield is a secure login system developed using Python Flask.

## Features
- User Registration
- Secure Login Authentication
- bcrypt Password Hashing
- SQLite Database
- Session Management
- Logout Functionality
- SQL Injection Protection
- Responsive User Interface

## Technologies Used
- Python
- Flask
- SQLite
- bcrypt
- HTML
- CSS

## Run the Project

pip install flask bcrypt

python init_db.py

python app.py
